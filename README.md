# How to use

## Install dependencies

```console
bundle install
```

## Update dependencies

```console
bundle update [gemname]
```

## see where a bundled gem is installed

```console
bundle info [gemname]
```

## Upgrade jekyll

```console
bundle update jekyll
```

```console
jekyll serve
```

```console
jekyll build
```
